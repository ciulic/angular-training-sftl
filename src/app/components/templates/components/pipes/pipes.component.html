Use pipes to transform strings, currency amounts, dates, and other data for display. Pipes are simple functions to use in template expressions to accept an input value and return a transformed value.

DatePipe: Formats a date value according to locale rules. 

  {{datePipe | date:'MMM d, y, h:mm:ss a'}}
  {{datePipe | date:'EEEE, MMMM d, y'}}
  {{datePipe | date:'h:mm:ss a zzzz'}}

UpperCasePipe: Transforms text to all upper case. 

  {{upperCasePipe | uppercase}}

LowerCasePipe: Transforms text to all lower case. 

  {{lowerCasePipe | lowercase}}

CurrencyPipe: Transforms a number to a currency string, formatted according to locale rules. 

  {{pricePipe | currency:'EUR'}}
  {{pricePipe | currency:'USD'}}

DecimalPipe: Transforms a number into a string with a decimal point, formatted according to locale rules. 

  {{decimalPipe|number}}
  {{decimalPipe|number:'1.0-4'}}
  {{decimalPipe|number:'2.0-5'}}
  {{decimalPipe|number:'3.1-5'}}

PercentPipe: Transforms a number to a percentage string, formatted according to locale rules. 

  {{percentPipe1|percent}}
  {{percentPipe2|percent}}
